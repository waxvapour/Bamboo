Ucase

Convert a string to upper-case.

Syntax
      Ucase(string)

Key
   string    A string or string expression.

Examples

Dim strDemo
strDemo = Ucase("12 Acacia Gardens")
Msgbox strDemo
Returns "12 ACACIA GARDENS "
