SetLocale

Set the value of the local computers LocaleID.

Syntax 
      SetLocale(LocaleID)

Key
   LocaleID    Decimal or String literal indicating the country.

Examples

'Set locale to the UK
SetLocale("en-gb")

'Set locale to Germany
SetLocale("de")
