DROP REMOTE SERVICE BINDING

Drop a remote service binding.

Syntax
      DROP REMOTE SERVICE BINDING binding [ ; ]

Key
   binding     The remote service binding to drop.

Server, database, and schema names cannot be specified.

Example

DROP REMOTE SERVICE BINDING MyBinding ;
