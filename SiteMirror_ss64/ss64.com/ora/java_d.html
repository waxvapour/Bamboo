DROP JAVA

Drop Java source.

Syntax:

   DROP JAVA {SOURCE|RESOURCE|CLASS} [schema.]object_name

SOURCE - drop a Java source schema object and all Java class schema objects derived from it.

RESOURCE - drop a Java resource schema object.

CLASS - drop a Java class schema object.

Related Oracle Commands:

JAVA - CREATE JAVA
JAVA - ALTER JAVA
