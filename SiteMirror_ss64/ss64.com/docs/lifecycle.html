Microsoft and Oracle Support Lifecycle dates

The Microsoft Support lifecycle begins when a product is released and ends when it is no longer supported.
These dates should inform your decisions about when to upgrade.

Product  Release date End of Mainstream support

  Windows 7 2009-07-22 2015-01-13
  Windows 8 2012-10-26 2018-01-09

  Windows 10, version 1511 2015-11-10 2017-10-10
  Windows 10, version 1607 2016-08-02 2018-04-10
  Windows 10, version 1703 2017-04-05 2018-10-09
  Windows 10, version 1709 2017-10-17 2019-04-09
  Windows 10, version 1803 2018-04-30 2019-11-12
Server      
  Windows Server 2003 R2 2005-12-06 2010-07-13
  Windows Server 2008 2008-02-27 2015-01-13
  Windows Server 2008 R2 Standard 2009-10-22 2015-01-13

  SQL Server 2008 R2 2010-04-21 2014-07-08
  SQL Server 2012 2012-03-06 2017-07-11

Oracle Database      
  Oracle 11.1 Aug 2007 Aug 2012
  Oracle 11.2 Sep 2009 Jan 2015
  Oracle 12.1 Jun 2013 Jul 2018

Microsoft Extended support, where available ( $$$ ) will add 5 years to the above support end dates.
Oracle Extended support, where available ( $$$ ) will add 3 years to the above support end dates.
Oracle Sustaining Support is indefinite for as long as you license the Oracle product.
