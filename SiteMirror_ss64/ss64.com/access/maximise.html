DoCmd.Maximise

Enlarge the active window, the current Access form or report.

Syntax
      DoCmd.Maximise

When an Access window is maximized, all other windows will also be maximized when you open them or switch to them.

The size of an Access Window will be constrained by the size of the window containing the MS Access application, unless the form is a popup form, in which case it can be moved outside the main window.

Pop-up forms are never maximized. To format a form as a PopUp, set the forms PopUp property to True.

Example

DoCmd.Maximise
